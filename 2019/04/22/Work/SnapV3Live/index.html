触发告警

接口说明

APP端下发触发告警指令(APP-->固件)

消息格式

{"dn":"B0:CE:18:FF:02:9D","type":"alarmTrigger","value":"0/1","time":"xxx"}

请求参数

参数名称 是否必须 描述
dn 否 所要控制的设备ID
type 是 “alarmTrigger”
value 是 “0”：关,”1”：开

响应结果，响应主题：snap/{mac}/status

响应格式

{"dn":"B0:CE:18:FF:02:9D","type":"alarmTrigger","value":"0/1/2","time":"xxx"}
参数名称 是否必须 描述
dn 否 所要控制的设备ID
type 是 “alarmTrigger”
value 是 “0”：关, “1”：开, “2”:告警中
