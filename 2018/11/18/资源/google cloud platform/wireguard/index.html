MAC版使用

  • 安装官网工具 brew install wireguard-tools
  • 把配置文件复制保存成 wrieguard.conf（conf结尾）的文件
  • command + shift + .找到/etc 创建 wireguard 文件夹
  • 将文件复制到wireguard
  • 关闭wg wg-quick down 指原网卡的名字 一般为 wgopenwrt
  • 开启wg wg-quick up wireguard
