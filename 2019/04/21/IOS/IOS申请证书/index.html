IOS证书申请流程

1.申请钥匙串文件

mac 电脑进入 Launchpad

-> 找到 钥匙串访问 启动

-> 运行后再左上角

-> 填写信息

-> 保存本地

2.开发者账号

打开苹果开发者中心（https://developer.apple.com）

3.企业账号申请流程

Apple ID 申请链接
注意：必须是企业域名的邮箱，不然后面不能往下走
密保问题一定要记住，修改资料或者密码都要用到
国家选择中国，除非你公司在其他国家

Join the Apple Developer Program

打开苹果开发者中心（https://developer.apple.com）
