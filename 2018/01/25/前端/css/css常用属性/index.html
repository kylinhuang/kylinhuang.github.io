常用属性

1 flex-direction 主轴方向

2 justify-content 主轴对齐方式（水平轴）

3 align-content 侧轴对齐方式（水平轴）

4 align-items 属性定义flex子项在flex容器的当前行的侧轴（纵轴）方向上的对齐方式。
